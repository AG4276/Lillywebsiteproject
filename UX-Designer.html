User experience designer is in charge of enhancing user satisfaction with a product by improving the usability, accessibility, and pleasure provided in the interaction with the product.

Nadia helps design the product to be easier to use and test the product to get feedback on how it could be better. She tests the product to see if it user friendly and if not finds a way to improve it. She talks to the developers about how the product can be designed to be easier to use.
